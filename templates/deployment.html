Deployment Guide for IT

Server Setup Checklist

Follow these steps to deploy the Intelligent Invoice Assistant on a production server.

1. Python Environment

Ensure Python 3.8+ is installed. Create a virtual environment and install the required libraries using pip:

pip install Flask pandas pdfplumber pillow pytesseract

2. Tesseract-OCR Engine

This application requires the Tesseract-OCR engine for processing scanned documents. Install it on the server:

  • Linux (Ubuntu/Debian): sudo apt-get install tesseract-ocr
  • Windows: Download and run the installer from the official Tesseract documentation.

3. Configure Tesseract Path in `app.py`

This is a critical step. Open the `app.py` file and locate the `CONFIG` section at the top. You MUST update the `TESSERACT_CMD` variable to point to the absolute path of the Tesseract executable on your server.

# Example for Linux:
# TESSERACT_CMD = r'/usr/bin/tesseract'

# Example for Windows:
# TESSERACT_CMD = r'C:\Program Files\Tesseract-OCR\tesseract.exe'

4. Folder Structure

The application expects a specific folder structure. Ensure these directories exist in the application root:

  • /uploads - For temporary storage of uploaded invoices. The application needs write permissions here.
  • /knowledge - Contains the CSV lookup files.
  • /templates - Contains the HTML files for the UI.

5. Running the Application

To run the application in a production environment, use a WSGI server like Gunicorn or Waitress.

Example with Gunicorn (Linux):

gunicorn --workers 4 --bind 0.0.0.0:8000 app:app

Example with Waitress (Windows):

pip install waitress
waitress-serve --host 127.0.0.1 --port 8000 app:app
